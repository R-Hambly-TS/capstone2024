About

This web page was developed for my Capstone Project for the Winter 2024 Semester.
This project represents a simulated distributed system for IoT devices.
The web page displays a table of 250 entries, once the entries reach the server,
they then get checked to see if they are abnormal or not. If they are abnormal
the will be flagged as such and then displayed in red on the web page table.

The application was built using Java Spring Boot on an Apache Tomcat Servlet.
The web design was created using the Thymeleaf Template Engine and a mix of
the following technologies:

  • HTML
  • Pure CSS
  • Font Awesome CSS
  • Bootstrap CSS

Planned Updates

Some planned updates for the Peanuts application are:
